This note exercises the content renderer that a real blog post would use.
It includes headings for the collapsible table of contents, math, code, and a masked spoiler.

## Math rendering

Inline math such as $E = mc^2$ should render through KaTeX.

Block math works as well:

$$
\int_0^1 x^2\,dx = \frac{1}{3}
$$

## Code fences

The renderer adds a language badge and a copy button to fenced code blocks.

```ts
type PostTheme = {
  title: string;
  themeColor: `#${string}`;
};

export function accentStyle(post: PostTheme) {
  return { "--post-theme": post.themeColor };
}
```

## Spoiler syntax

Use `:spoiler[hidden text]` when content should be blurred until hover or keyboard focus.
For example, :spoiler[this sentence starts hidden and becomes readable on interaction].
